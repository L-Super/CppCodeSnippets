| `/std:c++latest` | 查看文本            |

用途：判断使用的语言标准

可以直接打印
```c++
cout << __cplusplus << endl;
```
也可判断
```C++
// 支持C++11及以上
#if __cpluscplus >= 199711L
#else
#endif
```
